🚀 Ready to Get Started?

To use this extension in Microsoft Teams, you'll need to install it as a custom app.

Quick Setup:

  1. Deploy this app to a publicly accessible URL (HTTPS required)
  2. Create the Teams package using the provided scripts
  3. Upload to Teams via Apps → Upload a custom app
  4. Start voice messaging in any chat conversation!
